Animação de Alternância

Este código é um exemplo de uma página HTML que usa a biblioteca jQuery para animar um elemento div na página. Quando o botão "Mudar Animação" é clicado, o elemento div é animado para alternar entre ser visível e oculto usando o método slideToggle(). O código inclui o Bootstrap para fornecer estilos padrão para a página e um bloco de estilo interno que contém os estilos para o elemento div.

Animação de Clique do Mouse

Este código é um exemplo de uma página HTML que usa a biblioteca jQuery para detectar eventos de clique do mouse em um elemento p na página. Quando o botão do mouse é pressionado sobre o elemento p, o texto "Mouse Down" é adicionado ao elemento. Quando o botão do mouse é solto sobre o elemento p, o texto "Mouse Up" é adicionado ao elemento. Quando o botão "Limpar" é clicado, todos os elementos span dentro do elemento p são removidos. O código inclui o Bootstrap para fornecer estilos padrão para a página.
